# 琺瑯報價系統（Enamel Quote System）

**編號：** V_0002

**建立時間：** 2026-04-19

**說明：**
JFE 琺瑯板產品型錄系統，整合 Notion API 進行報價管理，並透過 Gmail 自動化寄送報價單給客戶。

**功能：**
- JFE 琺瑯板產品型錄
- Notion 報價系統整合
- 報價單 Gmail 自動化寄送
- Cloudflare Worker 部署版本
- 獨立運行版本（standalone）

**技術：**
- Notion API
- Gmail API
- Cloudflare Worker
- Python/Flask

**最後更新：** 2026-04-19
